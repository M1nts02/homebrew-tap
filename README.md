# M1nts02 Tap

## How do I install these formulae?

`brew install m1nts02/tap/<formula>`

Or `brew tap m1nts02/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "m1nts02/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
